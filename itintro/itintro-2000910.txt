- 1947 - 1960:
    - Transistor
    - Kõvakettad
    - Arvutite tööstuslik tootmine (IBM)
    - Fortran (esimene "päris" programmeerimiskeel)

    - Shockley Semiconductors 
        -> lagunes -> AMD ja Intel
    - Texas Instruments, Fairchild 
        - integrated circuit (blokk transistoreid ühel materjalil)
    - AT&T
        - Esimene modem

    - COBOL
        - Common Business Oriented Language
    - LISP
        - Disainitud AI programmide jaoks
        - Levinud edukalt NA, aga mitte EU
        -> Scheme
            -> JavaScript (ideed)

    - Esimene pela:
        - Space War!

- 1964:
    - IBM announces System/360
        - 6 interoperable arvutit ja 40 peripherali

- 1967:
    - IBM invents first floppy disk

- 1968:
    - Intel Corporation founded (Noyce, Moore)
        - Lahkusid Fairchild Semiconductorsist
    
    - Engelbart Stanfordis
        - Klaviatuur, hiir, window manager, remote collaboration jne

- 1969:
    - Esimene mikroprotsessor ehk CPU
    - Algselt oli eksperiment jaapani kliendi jaoks

- 1970:
    - Xerox avab Palo Alto Research Centeri (PARC)
    - Intel loob esimese kommertsiaalselt saadava DRAM chipi

- Photolitography
    -> Valgusallikas
    -> Mask (laseb ainult teatud kohtadest valgust läbi)
    -> Lääts, et koondada valgust (väiksemaks)
    -> Räniplaadile kantakse väike märge

- Trigger
    Tagasisidega gate, mille läbivoolu saab katkestada sõltumatult

- Mälu tüübid
    - Registrid (CPU state, flagid jne)
    - Cacheid (SRAM, Static RAM, ei pea vooluga täitma)
    - Tavamälu/RAM (DRAM, Dynamic RAM, peab vooluga täitma)
        - Kondensaatorid + transistorid (kui oli voolu, siis replenishime voolu perioodiliselt) 
    - Driveid

- Kolm eri arhitektuuri:
    - CISC
        - Complex instruction set
            - Intel ja AMD   
            - Väga palju kihte ja abstraktsioone
    - RISC
        - Reduced instruction set
            - ARM (mobiilid)
    - GPUd

- Oluline:
    - Integraalskeem (integrated circuit)
        - Ühel silikoonplaadil transistorid
    - Mikroprotesessor
        - Terve CPU ühel integraalskeemil
